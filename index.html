MAIMAI FESTIVAL TRANSITION — 纯 SVG + CSS 动画逐帧复刻

本项目使用 纯 SVG + CSS animation 复刻 舞萌 Maimai DX Festival 版本中的经典转场动画。基于官方录屏，还原了迪拉熊旋转缩放入场、底板缓出滑入、两角三角回弹、HOLD/SLIDE 条斜向飘入等全部图层效果。

✨ 不仅如此，这个转场动画可以直接用在你的网页中！只需引入一个 CSS 文件加上对应的 SVG 结构，即可为你的网站添加舞萌风格的页面过渡效果。欢迎在你的项目里使用这个转场！

常见问题

这个项目的动画来源是什么？

动画基于 @世界三傻 制作的舞萌转场视频，以 60fps 逐帧测量的方式提取每个图层的关键帧数据，再用 CSS @keyframes 和 animation 属性精确还原。@世界三傻 的转场视频本身来源于舞萌 Maimai DX Festival 官方录屏，在此一并感谢！

点击播放模式支持哪些操作？

页面加载后自动播放一次完整的入场动画（约 1.7 秒），之后处于静止状态。点击右上角"重播"按钮可以再次触发完整的入场 + 出场循环。入场阶段滴拉熊会旋转缩放从大到小落定，底板和三角图层依次滑入，HOLD/SLIDE 条带随机化的持续时间和延迟飘入。出场阶段各图层按逆向或飞离方向退场。

滚动驱动模式是如何工作的？

页面设计了 5 屏（500vh）滚动空间：前 100vh 展示第一页内容，100vh～300vh 之间由滚动位置驱动转场动画的时间轴（progress 0→1 映射动画的 0→2 秒），300vh～500vh 展示第二页内容。所有 CSS 动画被暂停（animation-play-state: paused），由 JavaScript 读取 scrollY 计算进度并手动设置 currentTime，实现完全线性可控的逐帧预览。

动画的技术细节有哪些？

核心技术点包括：

  • 滴拉熊（Chip）入场：rotate(-180deg → 0deg) 逆时针旋转 + scale(4.5 → 1.0) 缩放，配合透明度 0→1，共 10 个关键帧，时长 0.62s
  • 底板缓出：左上/右下底板分别从 -2400px/-1500px 和 +2400px/+1500px 以 cubic-bezier(0.22,1,0.36,1) 缓出至原点，时长 0.80s
  • 两角淡紫/白色三角：带 scale(1.06) 中心锚点扩展 + 回弹关键帧，确保回弹时远端边缘不露背景
  • HOLD/SLIDE 条：45° 对角滑入，每个 <use> 元素独立随机化 duration（0.55~0.80s）和 delay（0.17~0.23s），产生自然错落感
  • 出场动画：Chip 先内缩再外放淡出，底板沿入场方向继续飞离，两角三角先回弹再向角外飞出
为什么入场前有约 170ms 的空白？

参考视频前 10 帧（~167ms）画面完全空白，所有动画从第 10～12 帧才开始。因此 CSS 变量 --t0: 0.17s 作为全局起始延迟施加到所有图层，确保与参考视频的时间对齐。
