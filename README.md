# 舞萌 Festival 转场复刻 — Maimai DX Transition

使用 **纯 SVG + CSS 动画** 精确复刻 **舞萌 Maimai DX Festival** 版本中的经典转场效果。

## ✨ 体验转场

你可以通过以下两种方式体验这个转场：

- **[点击播放模式](https://gfdfdxc.github.io/maimai-transition/transition.html)** — 点击按钮触发完整的入场 + 出场动画，支持反复重播
- **[滚动驱动模式](https://gfdfdxc.github.io/maimai-transition/scroll.html)** — 鼠标滚轮控制动画进度，逐帧预览转场效果，模拟真实网页中的页面切换场景

## 🎯 关于项目

动画基于 [@世界三傻](https://space.bilibili.com/112891212) 制作的舞萌转场视频，以逐帧测量的方式提取每个图层的关键帧数据，再用 CSS `@keyframes` 和 `animation` 属性精确还原。纯 SVG + CSS 动画，无任何第三方依赖。

## 🛠 在你的网站中使用

这个转场动画可以直接应用在你的网页中，为你的网站添加舞萌风格的页面过渡效果。欢迎在你的项目里使用这个转场！
